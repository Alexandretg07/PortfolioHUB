# PortfolioHUB

Bem-vindo ao **PortfolioHUB**! 🚀

Este repositório foi criado como parte do "Desafio - Entrega Intermediária: Criação de Repositório com Versionamento". O objetivo deste espaço é centralizar, organizar e versionar todos os meus projetos acadêmicos e pessoais, construindo um perfil profissional robusto para apresentar ao mercado de trabalho.

## 📁 Estrutura do Repositório

- 🎓 **`projetos_academicos/`**: Trabalhos, pesquisas e projetos desenvolvidos ao longo do curso.
- 💡 **`projetos_pessoais/`**: Estudos independentes, projetos próprios e experimentações.
- 🌐 **Página Inicial**: Na raiz do repositório, você encontra um projeto web (`index.html` e `style.css`) que serve como portal de entrada para este portfólio.

## 🛠️ Tecnologias e Ferramentas

- **Git & GitHub**: Para controle de versão, colaboração e hospedagem.
- **GitHub Pages**: Utilizado para publicar a página web estática deste repositório.
- **HTML5 & CSS3**: Para o desenvolvimento da interface inicial.
